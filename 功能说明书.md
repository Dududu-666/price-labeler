## 一、商品管理（主页）

### 1. 扫码查价

条码输入框在页面左上角，自动聚焦。扫描枪对准商品条码，扫完后：

- **找到商品** → 表格自动显示该商品，弹出提示显示名称和售价
- **未找到商品** → 自动弹出"添加商品"窗口，条码已预填

扫码后输入框自动清空，可立即扫下一个。

### 2. 搜索商品

- **按条码搜索**：在扫码框输入条码，回车
- **按名称搜索**：在名称框输入关键词，自动模糊匹配
- **按分类筛选**：下拉选择分类，只显示该分类商品
- 条码搜索结果高亮后，点"清除条码搜索"恢复正常列表

### 3. 添加商品

点击右上角绿色 **「添加商品」** 按钮，填写：

| 字段 | 说明 | 必填 |
|------|------|------|
| 商品名称 | 如"可口可乐 330ml" | ✅ |
| 条码 | 扫码或手动输入 | ✅ |
| 分类 | 下拉选择 | 否 |
| 规格 | 如"500ml / 12瓶装" | 否 |
| 售价 | 零售价格（元） | 否 |
| 进价 | 进货价格（元） | 否 |

### 4. 修改价格（内联编辑）

在商品列表页，**直接点击价格数字**即可编辑：

- 点击售价或进价 → 出现输入框
- 输入新价格 → 按 **回车** 保存
- 自动记录变更历史（谁、什么时间、从多少改成多少）

### 5. 查看价格历史

每条商品右侧有 **「历史」** 按钮，点击弹窗显示：

- 售价变更记录（改了多少钱）
- 进价变更记录
- 变更时间和操作人

### 6. 删除商品

商品行右侧 **红色删除图标** → 确认 → 删除。

## 二、导入 Excel

点击右上角 **「导入 Excel」** 按钮。

### 支持的列名

系统自动识别以下列名（中英文、多种写法都行）：

| 字段 | 可识别的列名 |
|------|-------------|
| 名称 | 商品名称、产品名称、品名、name |
| 条码 | 条形码、SKU、barcode、编码 |
| 售价 | 零售价、标价、价格、单价、price |
| 进价 | 进货价、成本、采购价、cost |
| 分类 | 品类、类别、category |
| 规格 | 规格型号、包装规格、单位、spec |

### 导入流程

1. 拖拽或选择 Excel 文件（.xlsx / .xls）
2. 系统自动匹配列名，显示预览
3. 查看匹配结果（绿色标签 = 已匹配，红色 = 未匹配）
4. 点击 **「确认导入」**

### 导入规则

- 条码相同 → 更新该商品（不会重复）
- 条码不存在 → 新增商品
- 缺少条码的行 → 自动跳过

---

## 三、分类管理

左侧菜单点击 **「分类管理」**。

- **添加**：点击右上角"添加分类"，输入名称
- **编辑**：点"编辑"修改名称或排序
- **删除**：删除分类不会删除已有商品
- 修改分类名 → 所有使用该分类的商品自动更新

## 四、多设备使用

- 所有店员用同一个网址和账号
- 数据实时共享，任何人修改立刻生效
- 同时修改同一商品 → 后保存的覆盖先保存的
- 每台电脑的扫码枪都能正常使用
